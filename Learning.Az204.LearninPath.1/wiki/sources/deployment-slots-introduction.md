# Deployment Slots – Module Introduction

> [!source] learn.microsoft.com_en-us_training_modules_understand-app-service-deployment-slots_1-introduction.pdf

## Overview

This module covers [[deployment-slot|deployment slots]] in [[azure-app-service]] — a mechanism for previewing, testing, and deploying different environments.

> [!source] learn.microsoft.com_en-us_training_modules_understand-app-service-deployment-slots_1-introduction.pdf
> "The deployment slot functionality in App Service is a powerful tool that enables you to preview, manage, test, and deploy your different development environments."

## Learning Objectives

- Describe the benefits of using deployment slots.
- Understand how [[slot-swapping|slot swapping]] operates.
- Perform manual swaps and enable auto swap.
- Route traffic manually and automatically (see [[traffic-routing]]).
