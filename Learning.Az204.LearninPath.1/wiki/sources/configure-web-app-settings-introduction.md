# Configure Web App Settings – Introduction

> [!source] learn.microsoft.com_en-us_training_modules_configure-web-app-settings_1-introduction.pdf

## Module Overview

This unit introduces the **Configure Web App Settings** module. It sets the scope for understanding how [[azure-app-service]] handles environment configuration.

## Learning Objectives

After completing this module you will be able to:

- Create [[application-settings]] that are bound to [[deployment-slot|deployment slots]].
- Explain the options for installing SSL/TLS certificates (see [[security-certificates]]).
- Enable [[diagnostic-logging]] for your app to aid monitoring and debugging.
- Create virtual app to directory mappings (see [[path-mappings]]).

> [!source] learn.microsoft.com_en-us_training_modules_configure-web-app-settings_1-introduction.pdf
> "In App Service, app settings are variables passed as environment variables to the application code."
