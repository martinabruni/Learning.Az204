# Discover Microsoft Graph

> [!source] learn.microsoft.com_en-us_training_modules_microsoft-graph_2-microsoft-graph-overview.pdf

## Overview

[[microsoft-graph]] is the gateway to data and intelligence in Microsoft 365. It provides a unified programmability model to access data in Microsoft 365, Windows 10, and Enterprise Mobility + Security.

> [!source] learn.microsoft.com_en-us_training_modules_microsoft-graph_2-microsoft-graph-overview.pdf
> "Microsoft Graph is the gateway to data and intelligence in Microsoft 365."

## Key Claims

### Three Main Components of the Microsoft 365 Platform

1. **Microsoft Graph API** — single endpoint at `https://graph.microsoft.com`. Accessible via REST APIs or SDKs. Also includes services managing user and device identity, access, compliance, and security.

2. **Microsoft Graph connectors** — deliver **external data into** Microsoft Graph services and applications (incoming direction). Enhances Microsoft 365 experiences like Microsoft Search. Connectors exist for Box, Google Drive, Jira, Salesforce.

3. **Microsoft Graph Data Connect** — streamlines **secure and scalable delivery** of Microsoft Graph data to popular **Azure data stores**. Cached data serves as input for Azure development tools to build intelligent applications.
